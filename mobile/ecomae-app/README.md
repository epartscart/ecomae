# ecomae-app — native Android + iOS shell (Capacitor)

Thin native wrapper around the responsive web CP / ERP / storefront. Wraps the
live HTTPS site (same session-cookie login), adds a store listing + native push.

See `../../docs/MOBILE_APPS.md` for the full guide.

## Quick start
```bash
npm install

# choose one: ecomae-cp | tenant-cp | erp | storefront
npm run config:tenant-cp
npm run add:android        # first time
npm run add:ios            # first time (macOS)
npm run open:android       # Android Studio → signed .aab → Play Store
npm run open:ios           # Xcode → Archive → App Store
```

## Targets
Defined in `targets.json`. Override per build with env vars:
`APP_ID`, `APP_NAME`, `SERVER_URL`, `THEME_COLOR`, `BACKGROUND_COLOR`, `KEEP_HOST`.

## Notes
- `capacitor.config.json`, `android/`, `ios/`, `node_modules/` are generated and git-ignored.
- `www/index.html` is regenerated by `build-config.mjs` for the chosen target.
- Store submission needs your Google Play + Apple Developer accounts and signing certs.
